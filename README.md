- 경로 위치 표시 방식
- 경로 표시 기준의 변경 여부

- 절대 경로 방식
- Ex) 대한민국 서울특별시 ~~ 대호 빌딩 8층
- 출발지 위치에 상관없이 항상 찾아갈 수 있도록 표시
- Ex) 도메인주소 / 상세경로 => https://www.naver.com/blog/image.jpg

- Ex) 강남역 11번 출구에서 3분 걸어오면 대호빌딩
- 출발지 위치를 기준으로 표시
- Ex) 상세경로 => /blog/image.jpg || image.jpg
