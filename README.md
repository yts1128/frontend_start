`=(backtick)
```

* 시작태그만 있는 경우 : Empty Element 라고 부른다

```

> 포함관계 = (nested)
```
<html>
  <body>
    <h1> Heading </h1>
  </body>
</html>

```

> html-body-h1 관계
> 
> html : body 와의 관계 - 부모요소 / h1 과의 관계 - 조상요소
> 
> body : h1과의 관계 - 부모요소 / html 과의 관계 - 자식요소
> 
> h1 : body 와의 관계 - 자식요소 / html 과의 관계 - 자손요소
