### height , width

> - 블럭요소에서 가로길이 / 새로길이 적용시
> - auto : 요소의 기본 특성을 따라감 / default 값 : 적용하지 않은 상태와 같음
> - px : 픽스값으로 고정
> - % : 부모요소를 기준으로 특정 기준만큼 설정
> - 인라인 요소(<span>태그)에 가로길이 세로길이는 적용되지않음

### padding

> - padding-top , padding-right , padding-bottom , padding-left
> - padding : 10px 20px 30px 40px; (상/하/좌/우 따로)
> - padding : 10px 20px 30px; (상 / 좌우 / 하)
> - padding : 10px 20px; (상하 / 좌우 로 나뉨)
> - padding : 10px ; (네 면 똑같이 적용)

### margin

> - 마진 겹침 : 박스가 상하 대칭이 되어있을 때 사이 margin이 한쪽만 적용되는 현상
> - 두 박스 모두 마진 적용 하지않고 한 쪽에만 마진을 적용
